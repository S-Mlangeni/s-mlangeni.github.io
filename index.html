Single-Page Website

A web application that provides the users with information about their personal trainer’s training programs, testimonials, and contact info. Users can leave the trainer a message with their details on the app’s contact form, which the trainer can then use to contact them back. The contact form is connected to a MySQL database. The app is fully responsive which allows it to adjust and restructure based on the screen size/resolution, thus improving the user experience. The app also features a responsive navigation bar which, through the use of CSS animation, smoothly transitions in and out of the screen at the click of a button, thus adding a modern feel to the app.

The front- and back-end of the application were developed using HTML, CSS, and React JS, and the Node JS framework, Express JS, respectively. The two ends connect using API requests and are both hosted on the Heroku cloud platform. The app can be viewed using the “Demo” button below.

Login and Registration Form

A web application that only allows registered users to access the first and second page through providing valid credentials (i.e. their registered details). The user’s credentials are stored in a MySQL database when registering and retrieved when the user logs in. The application is secured as it does not allow access to the internal pages (first and second page) through URL manipulation and use of browser features (i.e. the back or forward button) without valid credentials, and it does not allow continual access to internal pages in another browser (in the case the URL is accessed in multiple browsers) once the user has logged out.

The application was developed using Java EE which featured Java servlets, JSP files, and CSS. It is hosted, along with its database, on Amazon Web Services (AWS) and can be viewed using the “Demo” button below.
